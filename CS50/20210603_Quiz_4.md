### 08. 아래 코드와 같이 피라미드 쌓기를 재귀적으로 작성한 코드에서, h 값으로 3이 입력되었을 때 draw 함수는 총 몇 번 호출될까요?

```
void draw( int h )
{
	if (h == 0)
	{
		return;
	}
	draw(h - 1);

	for(int i = 0; i < h; i++)
	{
		print("#");
	}
	printf("\n");
}
```

- h = 3일 때, 1회차에 1번 호출, 2회차에 h = 2가 되며 1번 호출, 3회차에 h = 1이 되며 1 번 호출 4회차에 0 되면서 리턴.
